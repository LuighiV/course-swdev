Información del curso

Descripción

Este curso es una iniciativa para efectuar una introducción a herramientas, conceptos y tecnologías empleados en el desarrollo de software incluyendo componentes de DevOps, Backend y Frontend.

Estructura del curso

El curso se divide en los siguientes temas.

  1. Scripting en Linux: Bash

  2. Control de versiones: Uso de Git / GitHub

  3. Gestión de servicios: Docker / Docker-compose

  4. Bases de datos: PostgreSQL

  5. Desarrollo en backend: ExpressJS

  6. Desarrollo en frontend: ReactJS

  7. Despliegue en la nube: Google Cloud

Comienza el curso

Para comenzar con el curse puede dirigirse a la página de Contenidos donde se encuentran los temas desarrollados hasta la fecha.
