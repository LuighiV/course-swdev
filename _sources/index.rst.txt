Información del curso
---------------------

Descripción
~~~~~~~~~~~

Este curso es una iniciativa para efectuar una introducción a herramientas,
conceptos y tecnologías empleados en el desarrollo de software 
incluyendo componentes de  DevOps, Backend y Frontend. 

Estructura del curso
~~~~~~~~~~~~~~~~~~~~

El curso se divide en los siguientes temas.

#. Scripting en Linux: Bash

#. Control de versiones: Uso de Git / GitHub

#. Gestión de servicios: Docker / Docker-compose

#. Bases de datos: PostgreSQL

#. Desarrollo en backend: ExpressJS

#. Desarrollo en frontend: ReactJS

#. Despliegue en la nube: Google Cloud

Comienza el curso
=================

Para comenzar con el curse puede dirigirse a la página de :doc:`Contenidos
</contents/index>` donde se encuentran los temas desarrollados hasta la fecha.
